# Add Binary

Difficulty: EASY.

[View this problem on Leetcode](https://leetcode.com/problems/add-binary/)

## Description

Give two binary `a` and `b`, return _their sum as a binary string_.

## Examples

**Input:** a = "11", b = "1"
**Output:** "100"

**Input:** a = "1010", b = "1011"
**Output:** "10101"
